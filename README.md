# Build a Space X clone

第一部分：本地构建一个新的专案
```
rails new RocketzCorp
cd RocketzCorp
git init
git add .
git commit -m "first commit"
```
---

第二部分：本地查看新的专案
```
atom .
rails server
http://localhost:3000/
```
---

第三部分：云端构建一个新的专案
```
git remote add origin https://github.com/shenzhoudance/RocketzCorp.git
git push -u origin master
```
---
